Dome commands

All dome-related commands begin with:

o.dome.

Dome shutter

1) Open the shutter:

o.dome.open_shutter()

2) Close the shutter:

o.dome.close_shutter()

3) Get the current shutter state (output: Open|Closed):

o.dome.get_shutter_state()

Dome orientation

1) Rotate the dome clockwise:

o.dome.rotate_cw()

2) Rotate the dome counter-clockwise:

o.dome.rotate_ccw()

3) Stop dome rotation:

o.dome.stop()

4) Rotate the dome to the given azimuth:

o.dome.rotate_to(azimuth)

Here azimuth is the azimuth angle (float number) value starting with 0 at the north clockwise in eastern direction.

5) Get the dome's slit center azimuth:

o.dome.get_azimuth()

6) Start continuous synchronization of the dome with the mount/telescope:

o.dome.sync()

7) Permanently stop synchronization of the dome with the mount/telescope:

o.dome.stop_sync()

8) Check whether the dome is in permanent synchronization with the mount/telescope (output: True|False):

o.dome.is_in_sync()

Dome sensors

1) Get humidity:

o.dome.get_humidity()

2) Get humidity temperature:

o.dome.get_humidity_temperature()

3) Get pressure:

o.dome.get_pressure()

4) Get temperature inside the dome:

o.dome.get_temperature_in()

5) Get temperature outside the dome:

o.dome.get_temperature_out()

Other dome commands

1) Check the current state of the dome driver (output: online|standby|suspend|offline):

o.scope.dome.state

2.1) Get the current offset (in degrees) of the dome from the mount/telescope:

o.scope.dome.offset

2.2) Set new offset (in degrees) of the dome from the mount/telescope:

o.scope.dome.offset(degrees)

Here degrees is the new offset angle (float number) value either positive or negative.

3.1) Check the current azimuth encoder resolution:

o.scope.dome.resolution

3.2) Set new azimuth encoder resolution:

o.scope.dome.resolution(resolution)

Here resolution is the new azimuth encoder resolution value.
