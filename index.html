Installation under Ubuntu

If you plan to use a FLI CCD camera along with the INDI Library driver, your computer must have Ubuntu version 15.10 or higher (check requirements at indilib.org website).

Requirements for installation/usage

Hardware: RAM: ; disk space: ; processor: .
Software: Python => ver., APEX => ver., astropy => ver., numpy => ver., scipy => ver.; star catalogs: e.g. Tycho.

Installation of dependencies: astropy, numpy and scipy

To make the installation of these dependencies simple, first install the python-pip package:

sudo apt-get install python-pip

Then install all three dependencies by one command:

sudo pip install -U astropy numpy scipy

Installation of dependencies: APEX

Please, refer to the APEX user manual.

Installation of FORTE's server part

Go to the folder where the tarball forte-0.15.0.tar.gz is stored and unpack it:

tar xvzf forte-0.15.0.tar.gz

A new folder forte-0.15.0 will be created with many files unpacked into it. Change to that folder:

cd forte-0.15.0

Installation of FORTE's client part

Go to the folder where the tarball forte_client-0.15.0.tar.gz is stored and unpack it:

tar xvzf forte_client-0.15.0.tar.gz

A new folder forte_client-0.15.0.tar.gz will be created with many files unpacked into it. Change to that folder:

cd forte_client-0.15.0.tar.gz
