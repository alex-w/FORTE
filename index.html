Installation under Ubuntu

Notice 1: FORTE (Facility for Operating Robotic TElescopes) is distributed under a proprietary license for ISON members only.

Notice 2: If you plan to use a FLI CCD camera along with the INDI Library driver, your computer must have Ubuntu version 15.10 or higher (check requirements at indilib.org website).

Notice 3: Currently, this manual describes FORTE command line usage only and doesn't describe FORTE GUI and its installation dependencies.

Installation of dependencies: build-essential, gfortran and python-dev

Install all packages:

sudo apt-get install build-essential gfortran python-dev

Installation of dependencies: AstroPy, NumPy and SciPy

To make the installation of these dependencies simple, first install the python-pip package:

sudo apt-get install python-pip

Then install all three dependencies:

sudo pip install -U astropy numpy scipy

Installation of dependencies: Tycho-2 star catalogue

Create the folder Tycho-2 anywhere in the user's home directory, e.g.:

mkdir ~/catalogs/Tycho-2

And put there all four files of the Tycho-2 catalogue: index.dat, suppl_1.dat, suppl_2.dat and tyc2.dat.

Installation of dependencies: APEX

Please, refer to the APEX user manual (soon online!) for its installation.

If not yet available, create the folder .Apex in the user's home directory:

mkdir ~/.Apex

If not yet available, create in the .Apex folder the file apex.conf with any ASCII text editor, and add or edit the following entries in it filling in your Tycho-2 folder's path, observing site's latitude (positive for North), longitude (positive for East) - both in decimal degrees, and altitude, e.g.:

[apex.catalog.tyc2_reader]
cat_path = ~/catalogs/Tycho-2
astrometric_priority = 10

[apex.sitedef]
latitude = 40.817566
longitude = 15.463387
altitude = 1256.21

Installation of FORTE server

Go to the folder where the tarball forte-0.15.0.tar.gz is stored and unpack it:

tar xvzf forte-0.15.0.tar.gz

A new folder forte-0.15.0 will be created with subfolders and files unpacked into it. Change to that folder:

cd forte-0.15.0

Build it:

python setup.py build

Install it (notice the sudo command!):

sudo python setup.py install

Installation of FORTE client

Notice: Installation of FORTE client must be done only, if you plan to use FORTE GUI; for FORTE command line usage instalation of FORTE server is sufficient.

Go to the folder where the tarball forte_client-0.15.0.tar.gz is stored and unpack it:

tar xvzf forte_client-0.15.0.tar.gz

A new folder forte_client-0.15.0.tar.gz will be created with subfolders and files unpacked into it. Change to that folder:

cd forte_client-0.15.0.tar.gz

Build it:

python client_setup.py build

Install it (notice the sudo command!):

sudo python client_setup.py install
